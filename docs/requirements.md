# PosterAI - AIポスター生成アプリ 要件定義書

## 🎯 プロジェクト概要

**PosterAI**は、AIを活用してイベント用ポスター、店内掲示用デザイン、SNSサムネイルなどを簡単に生成できるWebアプリケーションです。ユーザーは必要な情報を入力するだけで、プロフェッショナルなデザインのポスターを数秒で作成できます。

## 📋 主要機能

### 1. ポスター生成機能（コア機能）

#### 入力項目
- **用途選択**: イベント用 / 店内掲示用 / SNSサムネイル / その他
- **テイスト選択**: モダン / ポップ / エレガント / ミニマル / ビンテージ / プロフェッショナル
- **レイアウト選択**: 複数のレイアウトテンプレートから選択
- **メインカラー選択**: カラーピッカーまたはプリセットカラーパレット
- **メインタイトル**: ポスターのメインテキスト
- **サブタイトル**: サブテキスト（オプション）
- **フリーテキスト**: 追加で入れたいテキスト
- **フォント選択**: タイトル用およびサブテキスト用のフォント
- **サンプル画像アップロード**: テイスト、フォント、色遣いの参考画像（オプション、AI生成の参考用）
- **パーツアップロード**: 実際に生成画像に含めたいロゴやアイコン等の素材（オプション）

#### 出力機能
- **AIデザイン生成**: 入力情報を基にAIがポスターデザインを生成
- **出力サイズ選択**: ユーザーが用途に応じてサイズを選択可能
  - A3サイズ（297×420mm、4961×7016px @600dpi）イベント用ポスター
  - A4サイズ（210×297mm、3508×4961px @600dpi）店内掲示用
  - Instagram投稿用（1080×1080px）正方形
  - Twitter/X投稿用（1200×675px）横長
  - YouTube サムネイル（1280×720px）
- **プレビュー表示**: 生成されたポスターのリアルタイムプレビュー
- **再作成機能**: 同じ条件でランダムに異なるデザインを生成（回数制限なし）
- **画像ダウンロード**: PNG/JPEG形式でのダウンロード
- **画像保存**: ユーザーアカウントに保存（認証後）

### 2. ユーザー認証機能（後期実装）

- **Google認証**: Google OAuth 2.0を使用
- **認証なしモード**: ログインなしでも基本機能は利用可能
- **認証後の追加機能**:
  - 作成したポスターの保存
  - マイギャラリー機能
  - 作成履歴の管理

### 3. ギャラリー機能（認証後）

- 過去に作成したポスターの一覧表示
- お気に入り機能
- 再編集・再ダウンロード

## 🎨 デザイン要件

### UI/UXの方針
- **直感的な操作**: 初心者でも迷わず使える設計
- **モダンなデザイン**: 洗練された美しいUI
- **レスポンシブ対応**: PC、タブレット、スマートフォンで快適に利用可能
- **高速なフィードバック**: 生成中のローディング表示

### レイアウトテンプレート
- **レイアウトバリエーション**（6種類）:
  1. 中央集中型：タイトルとイメージを中央に配置
  2. 左右分割型：テキストと画像を左右に配置
  3. 上下分割型：上部にタイトル、下部に詳細
  4. 対角線型：ダイナミックな斜め配置
  5. フレーム型：中央にフレームを配置
  6. フリーフォーム型：自由な配置

### フォント選択
- **日本語フォント**（Google Fonts）:
  - Noto Sans JP（ゴシック体）
  - Noto Serif JP（明朝体）
  - M PLUS Rounded 1c（丸ゴシック）
  - Kosugi Maru（手書き風）
  - Zen Maru Gothic（モダンゴシック）
- **英字フォント**（Google Fonts）:
  - Montserrat（モダンSans-serif）
  - Playfair Display（エレガントSerif）
  - Bebas Neue（インパクト系）
  - Dancing Script（手書き風）
  - Roboto（汎用Sans-serif）

### カラースキーム
- プライマリカラー: ダイナミックで創造性を感じさせる色
- セカンダリカラー: 視認性の高い補色
- ダークモード対応: 目に優しい暗めのテーマ

## 🛠️ 技術スタック

### フロントエンド
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn/ui または Radix UI

### バックエンド/サービス
- **AI画像生成**: nanobanana または他のAI画像生成API
- **認証**: Better Auth (Google OAuth)
- **ファイルストレージ**: Cloudflare R2（画像保存用）
- **データベース**: Turso（ユーザーデータ、保存履歴）
- **ホスティング**: Vercel

### 開発ツール
- **Editor**: Cursor (ボイスモード活用)
- **Version Control**: Git + GitHub
- **Package Manager**: npm

## ⚠️ 重要な制約事項

> [!IMPORTANT]
> - **決済機能は不要**: サブスクリプションや課金機能は実装しない
> - **無料で利用可能**: AI生成APIの無料枠内での運用を想定
> - **認証は任意**: ログインなしでも基本機能（生成とダウンロード）は利用可能

> [!NOTE]
> - Google認証は後期フェーズで実装予定
> - 初期リリースでは認証なしモードで運用開始可能
